# City Generator

City Generator is a web-based application that allows users to generate random city maps. With a simple and intuitive interface, users can customize the layout of their city, including the number of districts, the size of the city, and the type of terrain. The application uses advanced algorithms to generate realistic city maps, complete with roads, parks, and buildings.

City Generator is a great tool for urban planners, game developers, and anyone interested in city design. Whether you're looking to create a fictional city for a game or a real-world city for a planning project, City Generator can help you get started quickly and easily.

*Please Note that the generator in its current form is at a basic level and cannot yet generate complex maps.

To get started with City Generator, simply visit the website and start generating maps. No installation or registration is required.

## Map key

* Green: Parks and green spaces
* Gray: Roads and paths
* Brown/Red: Buildings and structures
